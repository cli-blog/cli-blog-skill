# Cli Blog Skill For Claude

Use the bundled `cli-blog/SKILL.md` when the user asks Claude to add or operate Cli Blog in a project.

Relevant tasks include:

- Add blog publishing to an app.
- Integrate the Cli Blog API, `@cli-blog/node`, or `@cli-blog/cli`.
- Fetch published posts, authors, media, categories, tags, sitemap XML, or feed XML.
- Publish Markdown content from scripts, CI, or backend jobs.
- Choose a safe integration pattern for Next.js, React, Astro, Remix, PHP, Laravel, Python, Django, Rails, Go, or another backend.

When using the skill:

1. Read `cli-blog/SKILL.md`.
2. Load only the needed reference file from `cli-blog/references/`.
3. Keep private API keys on trusted servers.
4. Prefer examples that the user can paste into their project.
5. Verify with a demo CLI command or a small read request when practical.
